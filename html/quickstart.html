Quickstart guide
1. Download Open Broadcaster Software (OBS Studio)

OBS Studio is free and open source software for video recording and live streaming.
It is the re-write of the widely used Open Broadcaster Software, to allow even more features and multi-platform support. Currently Linux, Mac and Windows Builds are available.

Download OBS Studio: https://obsproject.com/download

2. Stream configuration

  • Stream type: Custom Streaming Server
  • URL: rtmp://thallium.ddns.net/live
  • Stream Key: Any alphanumeric combination (this will also be yout title)

3. Output configuration

  • Output Mode: Advanced
  • Bitrate: 3500 kb/s is maximum bitrate that is allowed (this also depends on your upload speed)
  • Keyframe Interval: 2
  • CPU Usage Preset: depends on your CPU, but veryfast preset is usually OK
  • Profile: high
  • Tune: zerolatency

4. Video configuration

  • Base (Canvas) Resolution: Same resolution as your monitor
  • Output (Scaled) Resolution: Resolution of your stream (commonly 1280x720)
  • FPS: Higher FPS uses more CPU (30 FPS is usually fine)
